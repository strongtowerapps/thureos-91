## Reporting a Vulnerability
If you discover a vulnerability that compromises the integrity or privacy of data shielded by Thureos-91, please **open a new Issue** in this repository. 

To help us resolve the issue quickly, please include:
1. A clear title and a detailed description of the potential flaw.
2. Steps to reproduce the exploit (Proof of Concept).
3. The potential impact on data integrity or encryption.



## Our Response Process
Once an issue is reported:
1. We will acknowledge the report as soon as possible through the GitHub Issue comments.
2. We will evaluate the severity and validity of the finding.
3. If confirmed, we will work on a fix and release a new version.

## Responsible Disclosure
We appreciate your help in making Thureos-91 more secure. We ask that you provide as much detail as possible to ensure a swift resolution for all users of the protocol.
