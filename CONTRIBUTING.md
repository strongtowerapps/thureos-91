Thank you for wanting to improve this protocol! To maintain code integrity, we follow these rules:

1. **Direct pushes to `main` are not allowed.**
2. All changes must come from a **separate branch**.
3. Opening a **Pull Request** is mandatory.
4. All **Jest** tests must pass before requesting a review.
5. At least **one approval** from a maintainer is required.

We can make Thureos-91 a more robust standard!
